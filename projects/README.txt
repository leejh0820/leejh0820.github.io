📁 Project Media Files
======================

Drop your project images or videos here.
Supported formats: .png, .jpg, .gif, .mp4, .webm

Then update Projects.jsx to reference them:
  media: '/projects/your-file.mp4'
  mediaType: 'video'   // or 'image'

Examples:
  /projects/rl-robot.mp4
  /projects/korean-nlp.png
  /projects/recruit-tracker.png
  /projects/dooray-mcp.png
  /projects/humanoid.png
